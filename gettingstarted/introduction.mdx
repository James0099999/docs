The iGamingAce API is a collection of protocols and tools that lets casino platforms integrate with iGamingAce's gaming system and offer players a diverse range of content — including slots and live games — through a single RESTful interface. Whether you run a single casino site or manage a network of operators, this API gives you the controls you need to launch games, manage player wallets, and run promotional campaigns.

## Key features

### Flexible RTP settings

You can customize the Return to Player (RTP) percentage to suit your business needs. RTP is configurable for each game providers separately, with a range from **75% to 99%**. This gives you fine-grained control over game economics across your player base.

### Bonus and free spin feature

You can grant players a specific amount of bonus credit or free spins to drive new registrations and reward loyal users. The API supports both free spin bonuses and guaranteed-win bonuses, so you have flexibility in how you structure promotional campaigns.

### Multi-currency and multilingual support

The API supports multiple currencies and languages out of the box, letting you serve a global audience without building your own localization layer. Players see content in their preferred language and transact in their local currency, which reduces friction and improves engagement.

### Seamless integration

iGamingAce integrates directly with your existing platform. In Seamless Integration mode, your backend manages player balances by responding to real-time wallet callbacks — no manual intervention required. **_iGamingAce only supports Seamless Wallet API._**
